# 西胪莲藕电商管理系统

## 项目简介

西胪莲藕电商管理系统是一个前后端分离的电商平台，专为汕头西胪莲藕产业设计，支持农户、客户和管理员三角色权限管理。系统具有商品管理、订单管理、购物车、溯源查询、大数据可视化等功能，适合作为大数据专业课程设计或学科竞赛项目。

## 技术栈

### 后端
- Java 17
- Spring Boot 3.2.0
- MyBatis-Plus 3.5.5
- MySQL 8.0
- Spring Security 6.2.0
- JWT

### 前端
- Vue 3
- Vite 5.0.0
- Element Plus 2.4.4
- Pinia 2.1.7
- Vue Router 4.2.5
- ECharts 5.4.3
- Axios 1.6.0

## 核心功能

### 1. 农户端
- 莲藕商品上架、修改、上下架
- 订单查看、发货
- 个人销量统计

### 2. 客户端
- 浏览莲藕商品、分类查询
- 购物车管理
- 收货地址管理
- 下单、模拟支付
- 订单管理
- 溯源查询

### 3. 管理员端
- 用户管理
- 商品审核
- 订单管理
- 大数据可视化分析（销量、品类、农户排行、库存预警）

## 数据库设计

系统包含以下核心表：
- `sys_user`：用户表（农户/客户/管理员）
- `product_category`：商品分类表
- `lotus_root_product`：莲藕商品表
- `shopping_cart`：购物车表
- `user_address`：收货地址表
- `order_main`：订单主表
- `order_item`：订单项表
- `traceability_info`：溯源信息表

## 快速开始

### 1. 环境准备
- JDK 17+
- MySQL 8.0+
- Node.js 16+

### 2. 数据库初始化
1. 执行 `database.sql` 脚本创建数据库和表结构
2. 初始化数据已包含在脚本中

### 3. 后端启动
1. 配置 `application.yml` 中的数据库连接信息
2. 运行 `LotusShopApplication.java` 启动后端服务
3. 后端服务默认运行在 `http://localhost:8080/api`

### 4. 前端启动
1. 进入 `frontend` 目录
2. 安装依赖：`npm install`
3. 启动前端服务：`npm run dev`
4. 前端服务默认运行在 `http://localhost:3000`

### 5. 登录测试
- 管理员账号：admin / 123456
- 农户账号：farmer1 / 123456
- 客户账号：customer1 / 123456

## 功能使用说明

### 农户端
1. 登录后进入农户首页，查看农场信息和商品统计
2. 商品管理：添加、编辑、上下架商品
3. 订单管理：查看订单状态，处理发货
4. 销量统计：查看销量趋势和商品销量排行

### 客户端
1. 登录后进入客户首页，浏览推荐商品
2. 商品浏览：按分类查询商品，查看商品详情
3. 购物车：添加商品到购物车，修改数量，删除商品
4. 收货地址：管理收货地址，设置默认地址
5. 订单管理：查看订单状态，取消订单，模拟支付
6. 溯源查询：输入商品ID查询溯源信息

### 管理员端
1. 登录后进入管理首页，查看系统概览
2. 用户管理：添加、编辑、删除用户，修改用户状态
3. 商品审核：审核农户提交的商品，通过或拒绝
4. 订单管理：查看所有订单，更新订单状态
5. 数据统计：查看销量趋势、商品分类占比、农户销量排行、库存预警

## 注意事项

1. 系统使用JWT进行身份认证，登录后会在本地存储token
2. 所有数据库操作使用MyBatis-Plus实现，包括分页、条件查询等
3. 前端使用Element Plus组件库，界面美观易用
4. 大数据可视化使用ECharts实现，展示各种统计图表
5. 系统支持三角色权限控制，不同角色只能访问对应功能

## 项目特色

- 地方特产：汕头西胪莲藕，突出产地、种植户、采摘时间、溯源信息
- 大数据方向：统计分析+ECharts可视化，符合大数据专业
- 前后端分离：使用Vue3+Spring Boot架构，代码结构清晰
- 三角色权限：农户、客户、管理员，功能划分明确
- 可直接用于课设和比赛：完整的功能实现，符合课程设计要求
